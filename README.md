# Tome Dashboard — модульная структура

Это рефакторинг вашего единого файла React-страницы на ясно выделенные модули и компоненты.
Функциональность полностью сохранена, а код разбит по фичам и слоям.

## Структура

```
src/
  TomeDashboard.jsx                # корневой компонент страницы (точка входа)
  components/
    Header.jsx                     # заголовок основной области
    ModalTransfer.jsx              # модальное окно пополнения/вывода
    NavButton.jsx                  # кнопка бокового меню
    Sidebar.jsx                    # левое меню + блоки баланса/бота
  data/
    transactions.js                # мок-данные последних транзакций
    balanceSeries.js               # ряд баланса (на будущее, для графика)
  hooks/
    useTweenNumber.js              # хук плавной анимации чисел (баланс)
  tabs/
    HomeTab.jsx                    # главная: обзор + последние транзакции
    DealsTab.jsx                   # сделки (заглушка)
    StatsTab.jsx                   # фильтры и заглушка статистики
    DetailsTab.jsx                 # таблица реквизитов
    EmployeesTab.jsx               # таблица сотрудников
    SettingsTab.jsx                # настройки (безопасность + профиль)
    DisputesTab.jsx                # споры (заглушка)
  utils/
    constants.js                   # константы приложения (баланс, адрес, версия)
    format.js                      # утилита форматирования чисел
```

## Как подключить

1. Скопируйте папку `src` в вашу кодовую базу (или объедините с уже существующей).
2. Импортируйте компонент страницы там, где он вам нужен:
   ```jsx
   import TomeDashboard from './src/TomeDashboard.jsx';
   // ...
   <TomeDashboard />
   ```
3. Tailwind CSS иконки/классы — без изменений. Внешних зависимостей (кроме React/Tailwind) не требуется.

## Что улучшено

- **Разделение ответственности**: компоненты `Sidebar`, `Header`, `ModalTransfer` и табы по фичам.
- **Переиспользуемость**: кнопка навигации и утилиты форматирования вынесены.
- **Масштабирование**: легко добавлять новые вкладки/виджеты, не трогая остальной код.
- **Анимации**: анимация баланса перенесена в хук `useTweenNumber` с чистым API.
